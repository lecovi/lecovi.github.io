1. Clone the repo.
2. Checkout to source branch `src-leo`.
3. `./nikola.sh`.
4. Edit your crap.

## Run nikola

By default `nikola.sh` will run `auto`.

```bash
./nikola.sh [sub_command] [PORT]
```

If you wanna get more help use `./nikola.sh help`
