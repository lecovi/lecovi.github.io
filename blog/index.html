    1. Instalar y configurar nuestro entorno de Python para nuestro sitio.
    2. Instalar y configurar git como sistema de control de versiones distribuido.
    3. Crear nuestro sitio con Nikola.
    4. Configurar nuestra cuenta en GitHub para el uso de GitHub Pages.
