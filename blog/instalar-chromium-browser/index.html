Instalar Chromium browser

Leandro E. Colombo Viña

2013-03-10 12:27

Comments

Código fuente

Si querés tener en tu linux la versión libre de Chrome, tenés que instalarte Chromium. De hecho, Chrome está basado en Chromium sólo que le agrega los componentes no libres de Google.

Instalemos Chromium en 4 pasos

  1. Afortunadamente el Chromium se encuentra en los repositorios de nuestro sistema LMDE. Para poder instalarlo debemos ir al Menu --> Gestor de Software.
  2. En el cuadro de búsqueda escribimos "chromium".
  3. Hacemos doble click sobre el primer resultado que nos aparece (el mismo que vemos en la imagen anterior).
  4. Y apretamos sobre el botón de "instalar". Una vez finalizado el proceso, dentro de Menu --> Internet --> Chromium podremos encontrar el acceso a este navegador.

Instalamos en 1 paso

También podemos instalarlo vía la consola ejecutando:

$ sudo apt update && sudo apt install chromium-browser
